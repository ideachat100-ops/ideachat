Course Overview

Master the universal theory of graphic design. This course covers cognitive psychology principles, spacing dynamics, layout grids, color theory, and typographic design strategies used by elite agencies worldwide.

What you will learn:

  • Visual Hierarchy: controlling focal order on flyers, websites, and booklets.
  • Layout Grids: building balanced layout compositions using mathematical proportions.
  • Typography System: establishing scalable font hierarchies and pairing matching weights.
  • Color Harmony: understanding emotional psychology, contrast swatches, and color accessibility rules.
