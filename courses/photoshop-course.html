Course Overview

This class takes you step-by-step from layout zero to Adobe tool hero. We focus strictly on design implementation. You will learn the layout mechanics behind iconic logos, vector alignments, brand stationaries, and high-fidelity print grids.

What you will learn:

  • Photoshop adjustments: hue, saturation, lighting layers, and curves.
  • Illustrator tools: pen tool mastery, node anchors, bezier curves, and pathfinders.
  • Layout rules: alignment guidelines, safe margins, and grid frameworks.
  • Production handoff: print bleed setups, CYMK vs RGB, and SVG assets generation.
